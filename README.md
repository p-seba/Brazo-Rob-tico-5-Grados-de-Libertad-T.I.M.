# Brazo Robótico de 5 Grados de Libertad - T.I.M.
Proyecto del Taller Inicial de Mecatrónica, Facultad de Ingeniería, UNCuyo. 

En este repositorio encontrarás todos los archivos asociados a la realización y presentación final del proyecto.    

Aplicaciones utilizadas: 
* Autodesk Fusion
* KiCad
* Visual Studio Code (PlatformIO)

En la carpeta de **Código** se encuentra una recopilación de todos los programas utilizados durante las pruebas y el código final. 

En la carpeta de **Diseño 3D** están los archivos de modelado y dos renders. Todo el diseño fue realizado por el grupo, excepto por el componente de la garra, el cual es una modificación del diseño original de **Julián Herrera**, disponible en: https://grabcad.com/library/gripper-and-servo-mg90s-1 

En la carpeta **PCB** están subidos los archivos de KiCad utilizados para diseñar las conexiones finales.  

En la carpeta **Documentación** están subidos todos los trabajos prácticos asociados a la materia y el informe final presentado en la mesa de examen.
